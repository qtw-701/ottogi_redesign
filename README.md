# 오뚜기 리디자인

아래 작업물들은 국비지원교육을 받으며 제작한 결과물입니다.

- 교육명 : 이젠 아카데미 - [과정평가형]UXUI 디지털디자인 및 웹디자인기능사 자격취득반

- 교육기간 : 2020/05 ~ 2020/10
